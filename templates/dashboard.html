WhatsApp Setup Guide

Connect to WhatsApp via Twilio

Follow these steps to make the chatbot live on WhatsApp

1
Create a Twilio Account

Go to twilio.com and sign up for a free account. Get your Account SID and Auth Token from the Console dashboard.

2
Enable WhatsApp Sandbox

In Twilio Console → Messaging → Try it Out → Send a WhatsApp message.
Join the sandbox by sending a join code from your WhatsApp to +1 415 523 8886.

3
Set Your .env File
TWILIO_ACCOUNT_SID=ACxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
TWILIO_AUTH_TOKEN=your_auth_token_here
ANTHROPIC_API_KEY=sk-ant-xxxxxxxxxxxx
PORT=8000
4
Start the server + expose with ngrok
# Terminal 1 — start server
./start.sh

# Terminal 2 — get public URL
ngrok http 8000
5
Configure Twilio Webhook URL

In Twilio Console → WhatsApp Sandbox → set the When a message comes in URL to:
https://<your-ngrok-url>/webhook/whatsapp (POST method)

6
Test it!

Send "Hi" from your WhatsApp to the sandbox number.
The bot will greet you and guide you through registration.

🏥 Screening Thresholds

Ibu Hamil (Kemenkes RI)

  • LiLA ≥23.5 cm → Normal
  • LiLA 21.0–23.4 → KEK (Perhatian)
  • LiLA <21 cm → KEK Berat (Rujuk)
  • Tekanan darah ≥140/90 → Hipertensi (Rujuk)
  • Hb ≥11 g/dL → Normal
  • Hb 7–10.9 → Anemia Sedang
  • Hb <7 → Anemia Berat (Rujuk)

Anak Bawah 2 Tahun (WHO)

  • WAZ ≥-2 → Normal
  • WAZ -2 s/d -3 → Berat Kurang
  • WAZ <-3 → Gizi Buruk (Rujuk)
  • HAZ ≥-2 → Normal
  • HAZ -2 s/d -3 → Stunting
  • HAZ <-3 → Stunting Berat (Rujuk)
